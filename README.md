# AI Resume Analyzer

An AI-powered resume analysis tool that extracts skills, evaluates resume quality, and provides improvement suggestions.

## Tech Stack
- FastAPI
- React (Vite)
- Groq LLM API
- PyMuPDF

## Features
- Upload PDF resume
- Extract skills automatically
- AI-powered resume score
- Suggestions for improvement
- Modern UI dashboard

# AI Resume Analyzer

An AI-powered resume analysis tool that extracts skills, evaluates resume quality, and provides improvement suggestions.

## Tech Stack
- FastAPI
- React (Vite)
- Groq LLM API
- PyMuPDF

## Features
- Upload PDF resume
- Extract skills automatically
- AI-powered resume score
- Suggestions for improvement
- Modern UI dashboard

## Run Backend
- cd backend
- pip install -r requirements.txt
- uvicorn main:app --reload

## Run Frontend
- cd frontend/ai_resume_frontend
- npm install
- npm run dev
